# Graph-ARI

Graph-ARI is a research repository focused on graph-based time series forecasting for mortality associated with Acute Respiratory Infections (ARI) in Mexico.

## Abstract

This project investigates how Graph Neural Networks and related machine learning models can be used to forecast ARI-related mortality from spatiotemporal health and environmental data. The proposed workflow combines weekly mortality records with meteorological and demographic variables, including land surface temperature, precipitation, aerosol measurements, population, and municipal-level geographic information. By modeling both temporal behavior and spatial relationships across municipalities, the project aims to improve mortality trend prediction, support early warning analysis, and provide evidence that can help public health teams plan interventions and allocate resources more effectively.
